Прототип динамичной 2D-игры, разработанный на движке Unity с использованием языка программирования C#. Проект демонстрирует применение современных практик разработки, включая оптимизацию производительности, модульную архитектуру и гибкое управление игровыми системами.

Описание проекта

Проект представляет собой рабочий прототип игры, в котором реализованы ключевые геймплейные и технические механики:
1) Оптимизированная архитектура: использование паттерна проектирования Object Pool для динамического создания и переиспользования снарядов и врагов, что исключает просадки кадров и снижает нагрузку на процессор.
2) Модульная система волн: настройка параметров противников и логики их спавна вынесена в Scriptable Objects, что позволяет легко менять баланс и добавлять новые уровни без изменения кода.
3) Визуальные эффекты: частицы опыта реализованы через встроенную систему частиц Particle System, а также числа нанесённого урона через Object Pool.
4) Разделение звука: настройка аудиоканалов выполнена с помощью инструмента Audio Mixer, разделяющего фоновую музыку и звуковые эффекты для их независимой регулировки.

Правила игры

Основная цель: продержаться как можно дольше, отражая последовательные волны противников, которые постепенно усложняются.
Игровой процесс: игрок управляет персонажем, который может свободно перемещаться по локации. Враги появляются волнами из автоматического спавнера и движутся в сторону игрока.
Условие поражения: игра заканчивается, если здоровье персонажа опускается до нуля в результате столкновений с противниками.

Инструкция по управлению

Управление персонажем адаптировано под стандартную раскладку ПК:
Перемещение: клавиши "W", "A", "S", "D" (или стрелочки на клавиатуре) для движения вверх, влево, вниз и вправо.
Атака: производится автоматически в сторону взгляда игрока.
Пауза: клавиша "Esc" для вызова меню паузы.
